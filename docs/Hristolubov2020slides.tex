\begin{block}{Цель}
Решить задачу классификации моментов временного ряда.
\end{block}

\begin{block}{Задача}
Требуется предложить способ построения признакового пространства и решить задачу классификации на построенном пространстве.
\end{block}

\begin{block}{Метод}
Суперпозиция моделей локальной аппроксимации и отбор признаков.
\end{block}

\end{frame}

\begin{frame}{Постановка задачи}

\begin{block}{Данные}
Задан исходный временной ряд c метками классов: 
$$\mathfrak{D}=\{(d_i, y_i\}_{i=1}^M,$$ где $y_i\in Y$, $|Y|$~---~количество классов. Предполагается, что ряд квазипериодичен, причем известен максимальный период $T$.
\end{block}

\begin{block}{Модель}
Модель будет приближать отображение $R:\mathcal{I}\rightarrow Y$, где $\mathcal{I}=\{1,\ldots M\}$~---~моменты времени. Будем искать ее в виде суперпозиции:
$$\hat R(k) = b(\varphi(h(k)), \mathbf{w}),$$
где $h:\mathcal{I}\rightarrow \mathbb{X}$~---~отображение момента времени в сегмент $\mathbf{x}_k$, $\varphi:\mathbb{X}\rightarrow W$~---~процедура построения признакового описания сегмента, $\mathbf{w}$~---~вектор параметров модели.
\end{block}

\end{frame}

%----------------------------------------------------------------------------------------------------------
\begin{frame}{Процедура построения признакового описания сегмента}

\begin{block}{Локально-аппроксимирующая модель}
Модель $\hat{\mathbf{x}}$ назовем локально-аппроксимирующей моделью, если она приближает временной ряд на локальном сегменте $\mathbf{x}$:
$$\hat{\mathbf{x}}=\argmin\limits_{\hat{\mathbf{x}}}||\hat{\mathbf{x}}-\mathbf{x}||_2.$$
\end{block}

\begin{block}{Признакове описание сегмента}
В качестве признакового описания $\varphi(\mathbf{x})$ сегмента $\mathbf{x}$ берется вектор параметров локально-аппроксимирующей модели $\hat{\mathbf{x}}(\mathbf{v})$:
$$\varphi(\mathbf{x})=\argmin\limits_{\mathbf{v}}||\hat{\mathbf{x}}(\mathbf{v})-\mathbf{x}||_2.$$
\end{block}


\end{frame}

%----------------------------------------------------------------------------------------------------------
\begin{frame}{Решение}

\begin{block}{Первый этап}
Классифицируемые моменты времени задают локальные сегменты $x_k$ временного ряда.
\end{block}

\begin{block}{Второй этап}
Для полученных сегментов $\mathbf{x}_k$ строится признаковое описание $\varphi(x_k)$ с помощью локально-аппроксимирующих моделей.
\end{block}

\begin{block}{Третий этап}
После получения признакового описания каждого момента времени, решается задача классификации моментов времени ряда.
\end{block}

\end{frame}
%----------------------------------------------------------------------------------------------------------
\begin{frame}{Вычислительный эксперимент}

\begin{block}{Цели}
Решить задачи:
\begin{list_type}  
\item 1) классификации типа активности,
\item 2) классификации пола человека.
\end{list_type}

Изучить влияние решения одной задачи классификации на решение другой. Сравнить распределения параметров моделей, построенных для решения этих задач.
\end{block}

\begin{block}{Данные}
Для эксперимента берется шесть временных рядов в $39000$ временных моментов ($780$ секунд). Данные снимаются с акселерометра, которым пользуются четыре человека (двое мужчин и две женщины), которые выполняют подъем или спуск по лестнице (два типа деятельности).
\end{block}

\end{frame}
%----------------------------------------------------------------------------------------------------------
\begin{frame}{Влияние решения одной задачи на другую}

\begin{block}{Кросс-валидация}
Для оценки обобщающей способности моделей используется k-fold кросс-валидация на $\text{k}=13$ подвыборках.
\end{block}
